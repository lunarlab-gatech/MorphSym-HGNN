# Dataset Installation and Structure

Extract any dataset contents into this directory. The provided .gitignore file should prevent any of the data from being committed.


## PROPRIOCEPTIVE SENSOR DATASET FOR QUADRUPED ROBOTS
Please navigate to https://ieee-dataport.org/open-access/proprioceptive-sensor-dataset-quadruped-robots and login to access the data. 

## Cerberus Demo Dataset
Please navigate to https://drive.google.com/drive/folders/13GsFDaBkDrslOl9BfE4AJnOn3ECDXVnc in order to access the data. More information on the data can be found here: https://github.com/ShuoYangRobotics/Cerberus?tab=readme-ov-file .
